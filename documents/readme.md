# YourAverageGamer

## Project Overview
This project is a class assignment to design and build a website called **YourAverageGamer**, inspired by gaming news and review platforms like IGN. The focus is on **understanding user needs** to create a site that is visually appealing, intuitive, and useful for gamers.

Currently, we are gathering requirements and learning about **user-centered design principles** to guide the website layout, content, and interactive features.

---

## Project Goals
- Understand and analyze user needs for a gaming website  
- Design a website structure and user experience that appeals to gamers  
- Learn to integrate project management and feedback tools into web design  

---
